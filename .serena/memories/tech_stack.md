# Tech Stack

**Primary Language & Environment:**
- **PowerShell 7+:** The core scripting language for the CLI and its functionalities, running on Windows 11.

**AI Models & Providers:**
- **Ollama:** Used for local AI model inference. Available models include `llama3.2:1b`, `llama3.2:3b` (default), `phi3:mini`, and `qwen2.5-coder:1.5b` (coder model).
- **Anthropic Claude:** Integrated via API (`ANTHROPIC_API_KEY`) for advanced AI capabilities.

**Multi-Context Protocol (MCP) Tools Integration:**
- **Serena:** Facilitates symbolic code analysis.
- **Desktop Commander:** Handles system operations.
- **Playwright:** Provides browser automation capabilities.

**Other Noteworthy Technologies:**
- **Rust:** Mentioned as an example for parallel execution principles.
- **TypeScript:** Also mentioned as an example for parallel execution principles, likely indicating its use in some parts of the project or as a reference for best practices.
